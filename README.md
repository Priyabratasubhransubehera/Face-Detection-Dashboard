# 🎯 Smart Face Detection System

A modern web-based **Face Detection System** that uses real-time camera access to detect faces and manage user authentication. This project is designed to automate attendance and improve security using AI-based detection.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:

```bash
cd your-repo-name
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm start
```

---

## 🔧 Environment Setup

Create a `.env` file and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

---

## 🚨 Important Fix (Routing for Netlify)

To fix mobile and direct URL issues (like `/dashboard` or `/login`), create a file:

```
public/_redirects
```

Add this line:

```
/* /index.html 200

```
